# User Stories

## User Roles

- **Guest** — unauthenticated visitor; can browse but cannot post, comment, or upvote
- **User** — authenticated account holder; full read/write access to their own content

---

## Core User Stories

### Authentication

1. As a guest, I want to register with a username, email, and password so that I can create an account on DevConnect.

2. As a guest, I want to log in with my email and password so that I can access my account.

3. As a user, I want to log out so that my session is ended on shared devices.

---

### Posts

4. As a guest, I want to view all posts in the feed so that I can browse questions without signing up.

5. As a guest, I want to view a single post with its full content so that I can read the details and discussion.

6. As a user, I want to create a post with a title and body so that I can ask a question or share knowledge.

7. As a user, I want to edit my own post so that I can correct or update information after publishing.

8. As a user, I want to delete my own post so that I can remove content I no longer want visible.

9. As a new user, I want to see pre-seeded posts when I first open the app so that the feed does not feel empty.

---

### Comments

10. As a guest, I want to view all comments on a post so that I can follow the discussion.

11. As a user, I want to add a comment on a post so that I can answer a question or contribute to a discussion.

12. As a user, I want to delete my own comment so that I can remove something I posted by mistake.

---

### Tags & Discovery

13. As a user, I want to add tags to my post so that others can find it by topic (e.g. `#javascript`, `#sql`).

14. As a guest, I want to filter the feed by tag so that I can find posts relevant to a specific technology.

15. As a guest, I want to sort the feed by newest or most upvoted so that I can find recent or trending posts quickly.

---

### Upvotes

16. As a user, I want to upvote a post so that I can signal that it is helpful or well-written.

17. As a user, I want to remove my upvote from a post so that I can undo an accidental or changed vote.

---

### Feedback & Edge Cases

18. As a user, I want to see a validation error if I submit an incomplete form so that I know what information is missing.

19. As a user, I want to see an error message if an action fails so that I understand what went wrong.

---

## Stretch Goal User Stories

20. As a user, I want AI-suggested tags when I create a post so that I do not have to manually label my content.

21. As a user, I want a "Summarize Discussion" button on long threads so that I can quickly grasp the key answers without reading every comment.

22. As a user, I want a natural language search so that I can query posts conversationally (e.g. "How do I fix a memory leak in Node?").

23. As a user, I want a post drafting assistant that prompts me to add missing details (e.g. expected output, environment) so that my question gets better answers.
